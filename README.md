# ResumeFit

ResumeFit is an AI-powered resume analyzer that evaluates resumes, compares them with a job description, and provides personalized feedback to help users improve their resumes for job applications.

## ✨ Features

- 📄 Upload PDF Resume
- 🎯 Resume Score Analysis
- 🤝 Job Description Matching
- 💻 Technical Skills Detection
- ✅ Skill Implementation Analysis
- 📂 Project Extraction
- 🎓 Education Extraction
- 💡 Personalized Suggestions for Improvement

## 🚀 Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
